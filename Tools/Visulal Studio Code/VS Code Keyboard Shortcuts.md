## 1. Windows OS

| **Command**                      | **Description**   |
| ---------------------------------|:-----------------:|
| **Cntl** **Shift** **P** | Open palette|
| **Cntl** **M** **M** | Toggle commads |
| **Cntl** **Shift** **N** | Open a new window.|
| **Cntl** **+**| Zoom out|
| **Cntl** **-**| Zoom in|
| **Cntl** ****Shift** **`****| Show terminal|
| **Cntl** **B**| Hide file explorer when the explorer window|
| **Cntl** **Shift** **E**| Show file explorer|
